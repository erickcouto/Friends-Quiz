# Friends Quiz - React Native Quiz app Example

A Quiz app example with sound integration and timer

## Libraries

* React Navigation
* Styled Components
* Reduxjs/toolkit
* React Native Sound

## Usage

```
yarn install
```

Then run the project using react-native CLI

```
yarn android
```

To run iOS you need to run pod install inside ios folder

```
pod install
```

```
yarn ios
```
